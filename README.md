# Python-100-days-of-code
The following repository is a coding challenge for continuous 100 days. The intention is to get in track in this phase of unemployment.
Rules: 
1) Atleast code for 1 hour.
2) Stay commited and consistent.
3) Have patience
4) Don't copy/paste any code.
5) Try to solve on your own. If you face any problem, try to understand the problem and then solve it.
